# NEXCUT AI Landing Copy

## Hero

長尺動画から、ショート動画候補を一瞬で。

## Subcopy

NEXCUT AIは、字幕AIと音声解析を使って、動画から切り抜き候補を自動生成するクリップ作成ツールです。  
アップロードした動画や字幕ファイルから、ショート動画向けの候補を作成し、複数クリップをZIPで一括ダウンロードできます。

## One-Liner

字幕がある動画も、字幕がない動画も。NEXCUT AIが切り抜き候補を見つけます。

## Problem

長尺動画からショート動画を作るには、見どころ探しに時間がかかります。

- どこを切り抜けばいいか分からない
- 字幕を見ながら探すのが大変
- 音楽や字幕なし動画では判断が難しい
- 複数クリップを書き出す作業が面倒

## Solution

NEXCUT AIは、字幕AIと音声解析でハイライト候補を作成します。

- 字幕AIハイライトで内容から候補生成
- 音声ハイライトで字幕なし動画にも対応
- Clipごとにプレビュー確認
- 開始秒・終了秒を微調整
- ZIP一括生成でまとめて保存

## Features

### 字幕AIハイライト

`.txt`, `.srt`, `.vtt` の字幕ファイルを使って、AIが内容を理解し、切り抜き候補を生成します。ローカル版ではYouTube字幕にも対応しています。

### 音声ハイライト

字幕がない動画や音楽動画でも、音声の盛り上がりから候補を作成します。

### Clipプレビュー

各Clipを開始秒から終了秒まで再生し、切り抜き前に確認できます。

### ZIP一括生成

複数のMP4クリップをまとめて生成し、ZIPで一括ダウンロードできます。

## Target Users

- YouTube動画をShorts化したい人
- TikTokやInstagram Reels向けに動画を切り抜きたい人
- セミナーや講義動画から要点を切り出したい人
- 動画編集の下準備を短縮したい人
- SNS用ショート動画を効率よく作りたい人

## Public Version Direction

公開版では、動画アップロードを中心にします。

- 動画ファイルアップロード
- 字幕ファイルアップロード
- 字幕AIハイライト
- 音声ハイライト
- ZIP一括生成

YouTube URL取得は、ローカル版または実験機能として扱います。

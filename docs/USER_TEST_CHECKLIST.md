# NEXCUT User Test Checklist

## Purpose

This checklist is used to test whether a first-time creator can understand and complete the NEXCUT AI workflow.

The goal is not to prove that the product is perfect.

The goal is to find where the creator gets stuck.

## Test Goal

Please try to complete one short video workflow using NEXCUT AI.

Do not ask for help unless you are completely stuck.

## Test Flow

Please go through these steps.

- Open NEXCUT AI
- Check the first screen
- Try Quick Generate
- Upload a video
- Create or confirm one clip
- Generate post assets
- Generate auto captions
- Generate translated subtitles
- Create a subtitled video
- Check Preview Studio
- Save or export the result

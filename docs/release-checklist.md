# NEXCUT AI Release Checklist

## 方針

現在の構成では、公開版はアップロード動画を主軸にする。

- `/` はアプリ本体
- `/landing` は紹介ページ
- YouTube URL取得はローカル向け・実験機能として扱う
- 公開版では動画アップロード、字幕ファイル、音声ハイライト、字幕AIハイライトを中心にする

## 公開時の推奨構成

### 推奨

- 動画アップロード
- 字幕ファイルアップロード
- 音声ハイライト
- 字幕AIハイライト
- ZIP一括生成

### 注意

YouTube URL取得は、以下の理由で公開版の主軸にしない。

- 403 / 429 が出る可能性
- yt-dlpがホスティング環境で不安定になる可能性
- Cookie利用はセキュリティリスクがある
- YouTube側の仕様変更に影響されやすい

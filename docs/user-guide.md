# NEXCUT AI 使い方ガイド

NEXCUT AIは、動画からショート動画向けの切り抜き候補を作成し、複数のMP4クリップをZIPでまとめてダウンロードできるツールです。

## 基本の使い方

1. 動画ファイルをアップロードします
2. 字幕がある場合は、字幕ファイルをアップロードします
3. 字幕がある動画は「字幕AIハイライト」を使います
4. 字幕がない動画や音楽動画は「音声ハイライト」を使います
5. 候補をプレビューします
6. 必要に応じて開始秒・終了秒を調整します
7. 投稿素材やAI台本が必要な場合は、Assetsで生成します
8. Exportで出力形式を選び、「ZIP一括生成」でMP4クリップをダウンロードします

## Story Wizard UI

NEXCUT AIでは、操作を5つのSTEPに分けています。

1. Upload: 動画と字幕ファイルを追加
2. Analyze: 字幕AIまたは音声ハイライトで候補生成
3. Clips: Clip候補を確認し、開始秒・終了秒を調整
4. Assets: 投稿素材、自動字幕、翻訳字幕を作成
5. Export: 出力形式を選び、ZIPで一括保存

### STEP4 Assetsでできること

STEP4は、以下の3つのタブに分かれています。

- 投稿素材: 投稿タイトル、説明文、ハッシュタグ、サムネ案、AI台本を生成します
- 字幕: Clip範囲の音声から自動字幕を生成し、字幕付き動画を作成します
- 翻訳: 自動字幕をもとに、日本語・英語の翻訳字幕や二段字幕付き動画を作成します

スマホでも操作しやすいように、STEP表示や操作ボタンは画面幅に合わせて並び替わります。

## 対応している字幕ファイル

以下の形式に対応しています。

- `.txt`
- `.srt`
- `.vtt`

`.txt` の例：

````md
```txt
10 This is the opening moment
25 Something surprising happens here
40 This is the most important part
55 The conclusion starts here
```

## 字幕AIハイライト

字幕AIハイライトは、字幕の内容をもとにAIが切り抜き候補を作成する機能です。

※「字幕を要約する」は、読み込んだ字幕の内容を短くまとめる機能です。
長い動画の内容をすばやく確認したり、投稿タイトル・説明文・AI台本を作る前の参考にできます。

おすすめ用途：

- 解説動画
- インタビュー
- 講義
- トーク動画

## 音声ハイライト

音声ハイライトは、音の盛り上がりをもとに切り抜き候補を作成する機能です。

おすすめ用途：

- 音楽動画
- スポーツ動画
- 字幕がない動画
- 盛り上がりが音で分かる動画

## AI台本生成

AI台本生成は、Clip候補をもとにショート動画向けのナレーション台本を作成する機能です。

対応している長さ：

- 15秒
- 30秒
- 60秒
- 90秒

出力内容：

- Hook
- Script
- Ending
- Full Script

おすすめ用途：

- Shorts用ナレーション
- TikTok用台本
- Reels用原稿
- 投稿前の構成確認

## 画質について

NEXCUT AIでは、生成するクリップの画質をできるだけ保つため、元動画の映像を再圧縮せずに切り抜く方式を使っています。

メリット：

- 元動画に近い画質を保ちやすい
- 処理が速い
- サーバーへの負荷が軽い

注意点：

- 開始位置がほんの少しズレることがあります
- 一部の動画では、最初の一瞬が黒くなることがあります
- 元動画の画質が低い場合、生成されるクリップも低画質になります

## 9:16縦動画について

現在のNEXCUT AIは、元動画の比率を保った切り抜き生成を中心にしています。

今後、YouTube Shorts、TikTok、Instagram Reels向けに、9:16の縦動画出力へ対応予定です。

予定している流れ：

- 1080×1920の縦動画生成
- まずは中央クロップで対応
- 将来的に人物や顔を追尾するクロップへ拡張

## YouTube URL機能について

YouTube URLからの動画取得は、ローカル環境向けの実験機能です。

公開版では、以下の理由から動画アップロードを推奨します。

- YouTube側の制限で取得できないことがある
- 403 / 429 エラーが出ることがある
- Cookieを使う運用は安全性のリスクがある
- ホスティング環境ではyt-dlpが不安定になる可能性がある

## Gemini APIについて

字幕AIハイライトではGemini APIを使用します。

無料枠には上限があります。上限に達した場合は、時間をおいて再度試してください。

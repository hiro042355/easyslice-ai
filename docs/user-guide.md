# NEXCUT AI 使い方ガイド

NEXCUT AIは、動画からショート動画向けの切り抜き候補を作成し、複数のMP4クリップをZIPでまとめてダウンロードできるツールです。

## 基本の使い方

1. 動画ファイルをアップロードします
2. 字幕がある場合は、字幕ファイルをアップロードします
3. 字幕がある動画は「字幕AIハイライト」を使います
4. 字幕がない動画や音楽動画は「音声ハイライト」を使います
5. 候補をプレビューします
6. 必要に応じて開始秒・終了秒を調整します
7. 「ZIP一括生成」でMP4クリップをダウンロードします

## 対応している字幕ファイル

以下の形式に対応しています。

- `.txt`
- `.srt`
- `.vtt`

`.txt` の例：

````md
```txt
10 This is the opening moment
25 Something surprising happens here
40 This is the most important part
55 The conclusion starts here
```

## 字幕AIハイライト

字幕AIハイライトは、字幕の内容をもとにAIが切り抜き候補を作成する機能です。

おすすめ用途：

- 解説動画
- インタビュー
- 講義
- トーク動画

## 音声ハイライト

音声ハイライトは、音の盛り上がりをもとに切り抜き候補を作成する機能です。

おすすめ用途：

- 音楽動画
- スポーツ動画
- 字幕がない動画
- 盛り上がりが音で分かる動画

## 画質について

NEXCUT AIでは、生成するクリップの画質をできるだけ保つため、元動画の映像を再圧縮せずに切り抜く方式を使っています。

メリット：

- 元動画に近い画質を保ちやすい
- 処理が速い
- サーバーへの負荷が軽い

注意点：

- 開始位置がほんの少しズレることがあります
- 一部の動画では、最初の一瞬が黒くなることがあります
- 元動画の画質が低い場合、生成されるクリップも低画質になります

## YouTube URL機能について

YouTube URLからの動画取得は、ローカル環境向けの実験機能です。

公開版では、以下の理由から動画アップロードを推奨します。

- YouTube側の制限で取得できないことがある
- 403 / 429 エラーが出ることがある
- Cookieを使う運用は安全性のリスクがある
- ホスティング環境ではyt-dlpが不安定になる可能性がある

## Gemini APIについて

字幕AIハイライトではGemini APIを使用します。

無料枠には上限があります。上限に達した場合は、時間をおいて再度試してください。

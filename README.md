# NEXCUT AI

> AI動画切り抜きプラットフォーム

NEXCUT AIは、動画からショート動画向けのハイライトクリップを作成するためのAIツールです。

YouTube動画、アップロード動画、字幕ファイル、音声解析を使って、複数の切り抜き候補を作成し、MP4クリップをZIPで一括ダウンロードできます。

## 使い方ガイド

アプリ内の使い方ページで、基本操作、字幕ファイルの形式、音声ハイライト、画質に関する注意点を確認できます。

URL: `/guide`

## 主な機能

- 動画ファイルのアップロード
- YouTube動画の取得（ローカル環境向け）
- YouTube字幕の取得
- 字幕ファイル読み込み
  - `.txt`
  - `.srt`
  - `.vtt`
- Gemini APIによる字幕AIハイライト生成
- 字幕なし動画・音楽動画向けの音声ハイライト生成
- Clipごとのプレビュー再生
- AIによる投稿タイトル・説明文・ハッシュタグ生成
- AIによるサムネ文言・サムネ構成案生成
- AIによるショート動画台本生成
- 終了秒で自動停止するプレビュー
- Clipの開始秒・終了秒の手動編集
- ±5秒ボタンによる微調整
- 複数Clipの一括生成
- MP4クリップをZIPで一括ダウンロード
- ClipタイトルをZIP内ファイル名に反映

## 推奨ワークフロー

### アップロード動画の場合

1. 動画ファイルをアップロード
2. 字幕ファイル（`.txt`, `.srt`, `.vtt`）があればアップロード
3. 字幕がある場合は「字幕AIハイライト」を使用
4. 字幕がない場合や音楽動画の場合は「音声ハイライト」を使用
5. Clipをプレビューして確認
6. 必要に応じて開始秒・終了秒を調整
7. 「投稿素材生成」で投稿タイトル・説明文・ハッシュタグ・サムネ案を作成
8. 「AI台本生成」でショート動画向け台本を作成
9. 「ZIP一括生成」でダウンロード

### YouTube動画の場合

1. YouTube URLを入力
2. 動画を取得
3. 字幕がある場合は「YouTube字幕取得」
4. 「字幕AIハイライト」または「音声ハイライト」で候補生成
5. Clipをプレビューして確認
6. 「投稿素材生成」で投稿タイトル・説明文・ハッシュタグ・サムネ案を作成
7. 「AI台本生成」でショート動画向け台本を作成
8. 「ZIP一括生成」でダウンロード

## ローカルセットアップ

### 必要なもの

- Node.js
- ffmpeg
- yt-dlp
- Gemini APIキー

### インストール

```bash
npm install
```
### 環境変数

プロジェクト直下に `.env.local` を作成します。

```env
GEMINI_API_KEY=your_gemini_api_key
NEXT_PUBLIC_ENABLE_YOUTUBE=true
```

`NEXT_PUBLIC_ENABLE_YOUTUBE` はYouTube機能の表示切り替えに使います。

- `true`: YouTube URL取得・YouTube字幕取得を表示
- `false`: YouTube関連UIを非表示

公開版では、YouTube機能は不安定になる可能性があるため、`false` を推奨します。
